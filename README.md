# L10n Glossary

Terminology editor with consistency checking across translation files.

Built with GTK4/Adwaita. Part of the [Danne L10n Suite](https://github.com/yeager/debian-repo).

## Installation

### Debian/Ubuntu
```bash
sudo apt install l10n-glossary
```

### Fedora/RPM
```bash
sudo dnf install l10n-glossary
```
